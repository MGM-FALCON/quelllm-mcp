# Décisions architecturales (ADR) — quelllm-mcp

> Append-only. Avant tout changement archi, GREP ici. Toute nouvelle décision non triviale = un ADR.

## Format
```
### YYYY-MM-DD — ADR-NNN — <titre>
- **Contexte** : situation, contrainte
- **Choix retenu** : ce qu'on fait
- **Alternatives écartées** : X (raison), Y (raison)
- **Conséquences** : ce que ça implique (positif et négatif)
- **À revisiter si** : condition de remise en question
```
