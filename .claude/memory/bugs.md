# Bugs résolus — quelllm-mcp

## Format
```
### YYYY-MM-DD — <titre court>
- **Symptôme** : ce qu'on voit
- **Cause racine** : pourquoi
- **Fix** : ce qu'on a fait (commit/PR si possible)
- **Prévention** : test, garde-fou, doc à mettre à jour
```
